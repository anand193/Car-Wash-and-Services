# 🚗 Car Wash and Service Dashboard

## 📊 Overview
This project presents an interactive **Car Wash and Service Dashboard** designed to analyze vehicle data, sales performance, and service trends. It provides key insights into car distribution, fuel types, transmission types, and yearly sales.

---

## 📌 Features

- **Total Metrics Overview**
  - Total Cars: 50,000
  - Total Sales: 1.30M
  - Total Brands: 10
  - Total Fuel Types: 5
  - Total Models: 10

- **Fuel Type Analysis**
  - Diesel, Electric, CNG, Hybrid, Petrol comparison

- **Sales Trends**
  - Year-wise sales breakdown (2022–2024)

- **Transmission Analysis**
  - Manual vs Automatic distribution

- **Model Distribution**
  - Breakdown of cars by model (City, Creta, Duster, XUV500, Ecosport, Swift, etc.)

---

## 🛠️ Tech Stack

- **Data Visualization Tool**: Power BI / Tableau  
- **Data Processing**: Excel / CSV / SQL  
- **Version Control**: Git & GitHub
